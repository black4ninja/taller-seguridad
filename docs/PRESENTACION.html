Hack the Campus   // CTF STRIDE · 1:30h · individual

1. Prerequisites

  • Node.js 20+  node -v
  • Docker Desktop (para ZAP y Semgrep)
  • Git
  • Editor a elección (VS Code, vim, etc.)

Si npm install falla compilando better-sqlite3:
· macOS: xcode-select --install
· Linux: apt install build-essential
· Windows: VS con C++ tools

2. Setup (10 min)

  • git clone <repo-url>
  • cd hack-the-campus
  • npm install
  • npm start → :3000
  • npm test → 7 rojos

Si npm test se cuelga: es V5 (ReDoS).
Usa: npm test -- --testPathIgnorePatterns=V5

3. Tu arsenal (MobSF → web)

  • npm run scan:audit → SCA (10s)
  • npm run scan:semgrep → SAST (1m)
  • npm run scan:zap → DAST (2-3m)

Para competir basta 1 de los 3.
Prioriza npm audit.
